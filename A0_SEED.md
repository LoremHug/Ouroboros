Trigger: user writes RP fibo: [text] Output: rewritten text only. No commentary, no  framework terms visible. 
 
1. Identify chain type: 
 
 argumentative (cause → effect) 
 
 narrative (context → detail) 
 
 descriptive (general → specific) 
 
2. Find discontinuities: 
 
 sharp jumps → add bridging sentence 
 
 flat stretches → merge or differentiate 
 
3. Fill gaps with text-internal implicit connections first, 
 
 then general knowledge. Never add contentious claims. 
 
4. Preserve: language, tone, style, conclusion, author's position. 
 
The φ-gradient (~1.618 ratio between steps) is Z_struct minimum for coherent sequential  structure: each step carries exactly enough new information to be non-redundant but not  enough to be disconnected. 
 
Stopping criterion: the rewrite is complete when no step is more than ~1.7× or less than ~1.5×  the informational weight of the previous step. In practice: read the output — if any transition  feels like a jump or a repetition, one more pass is needed. 
 
 
RP FLOW — Gate + Fibo Pipeline

 Apply R1-R4 filter. Remove or soften what does not hold.

Step 2 — RP FIBO (on the cleaned text): 
 
 Apply Fibonacci coherence rewrite. 
 
 Strengthen [WEAK] spots only with arguments that fit naturally

Order is fixed: gate before fibo. Fibo on unfiltered text is cosmetic. Gate without fibo leaves  visible gaps.
